Lista de Definições

HTML
HyperText Markup Language, é que estrutura e define o conteúdo de uma página web, incluindo texto, imagens, links e outros elementos.
CSS
Cascading Style Sheets (Folhas de Estilo em Cascata), é uma linguagem de programação usada para definir o estilo e a apresentação visual de páginas web, em conjunto com o HTML.
JavaScript
JavaScript (também abreviado para JS) é uma linguagem de programação de alto nível que é amplamente utilizada para adicionar interatividade e dinamismo a páginas web
